Rubric: Full Stack MEVN Sports Scheduling Application

Criteria / Marks Description Level 1 Level 2 Level 3 Level 4
Backend (20) Assess the functionality and efficiency of the backend implementation using Node.js for RESTful API development and MongoDB for database management within the MEVN stack. Limited understanding of backend technologies, inefficient API design, and poor database management. Basic understanding of backend technologies, functional API with some inefficiencies, and basic database management. Adequate understanding of backend technologies, functional API with minor inefficiencies, and organized database management. Exceptional understanding of backend technologies, highly efficient API design, and optimal database management.
Frontend using Router (15) Evaluate the implementation of frontend routing using Vue Router to create a seamless user experience and navigate between different views/pages of the application. Limited or no implementation of Vue Router, resulting in poor navigation and user experience. Basic implementation of Vue Router with some issues in navigation and user experience. Adequate implementation of Vue Router, providing satisfactory navigation and user experience. Seamless integration of Vue Router, providing smooth navigation and excellent user experience.
Look and Feel (15) Evaluate the user interface design and overall aesthetic appeal of the application, including layout, color scheme, typography, and visual elements. Poor design choices, lack of consistency, and unappealing interface, impacting user experience negatively. Basic design elements with some inconsistencies and average user experience. Adequate design elements with minor inconsistencies and satisfactory user experience. Exceptional design elements, attention to detail, and aesthetically pleasing interface, enhancing user experience significantly.
Authentication and Authorization (10) Assess the implementation of user authentication and authorization features, ensuring secure access to administrative functionalities and protecting user data within the MEVN architecture. No or inadequate implementation of authentication and authorization features, leaving the application vulnerable to security threats. Basic implementation of authentication and authorization features, with some vulnerabilities or gaps in security measures. Adequate implementation of authentication and authorization features, ensuring secure access to administrative functionalities and user data. Robust implementation of authentication and authorization features, providing secure access and protecting user data effectively.
Integration (10) Evaluate the integration of third-party APIs (such as Google Maps) and their effective utilization to enhance the application's functionality and user experience. No integration of third-party APIs, severely limiting application functionality and user experience. Limited integration of third-party APIs, resulting in functionality issues or poor user experience. Basic integration of third-party APIs, with some limitations in functionality or user experience. Seamless integration of third-party APIs, enhancing application functionality and providing an excellent user experience.
Deployment and Database (10) Evaluate the deployment of the application to the cloud and the presence of data in the database, ensuring that the application is accessible online and functional. Application is not deployed to the cloud, or database is empty, rendering the application inaccessible or non-functional. Application is deployed to the cloud, but database is empty or lacks sufficient data, limiting functionality and usability. Application is deployed to the cloud, and database contains some data, providing basic functionality and usability. Application is successfully deployed to the cloud, and database contains sufficient data, ensuring full functionality and usability.
Proper Use of Vue.js API (10) Assess whether the Vue.js options API or composition API is properly utilized to create the frontend components and manage state within the application. Limited or improper use of Vue.js API, resulting in poorly structured components and inefficient state management. Basic use of Vue.js API, with some issues in component structure and state management. Adequate use of Vue.js API, with well-structured components and satisfactory state management. Excellent use of Vue.js API, with properly structured components and efficient state management.
In-Class Work on Project (5) Assess the contribution to the planning, development, and production of the project as a team member, focusing on backend and frontend development tasks within the MEVN stack. Did not contribute to the project, demonstrating a lack of teamwork and effort. Contributed minimally to the project, demonstrating poor teamwork and effort. Contributed adequately to some aspects of the project, with some issues in teamwork or effort. Actively contributed to all aspects of the project, demonstrating exceptional teamwork and effort.
Meeting Project Goals and Deadlines (5) Evaluate the ability to meet project goals and deadlines by managing individual time effectively during backend and frontend development phases within the MEVN framework. Severely failed to meet project goals and deadlines, indicating a complete lack of time management and organization skills. Failed to meet project goals and deadlines, demonstrating poor time management and organization skills. Mostly met project goals and deadlines, with occasional issues in time management or organization. Consistently met project goals and deadlines, demonstrating excellent time management and organization skills.
